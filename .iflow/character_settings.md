# 人物设定档案

## 核心铁三角

---

### 姜辰 (Jiang Chen) - 镇灵师末代传人

**基础信息**
- **年龄**：28岁
- **身高**：178cm
- **体重**：68kg
- **外貌**：清瘦，皮肤苍白，戴金丝眼镜，学者气质。左手手背有淡青色血咒纹路（问玉玦碎裂后显现）
- **身份**：古玩店「藏玉阁」店主 / 没落镇灵师家族唯一传人

**性格特征**
- **核心特质**：冷静、理智、知识渊博，轻微强迫症与洁癖
- **优点**：思维缜密，记忆力超群，对《镇灵谱》倒背如流，危机中能保持理性分析
- **缺点**：实战经验为零，体能差，洁癖在墓下环境造成严重心理负担
- **口头禅**："根据《镇灵谱》记载……" / "这不合逻辑"
- **恐惧**：身体失控、污秽、未知生物触碰

**背景故事**
- 镇灵师姜家第47代传人，家族因"只镇不取"的祖训而没落
- 20年前爷爷姜穆在玄甲墓失踪，留下《镇灵谱》真本和问玉玦
- 父母早逝，由爷爷旧友抚养长大，从未下过地
- 三个月前血咒首次发作，左手出现黑色经络，身体定时僵化

**核心动机**
- **表层**：解开血咒，活下去
- **深层**：寻找爷爷失踪的真相，弄清"九幽灵"的本质
- **潜在**：打破祖训，重新定义镇灵师的使命

**特殊能力**
1. **灵怨感知**：问玉玦破碎后觉醒鬼眼，无需外物即可感知灵怨浓度与流向
2. **血咒镇压**：血液可绘制破煞符，中和灵怨（但会加剧自身侵蚀）
3. **鬼手状态**（血咒过载）：
   - **触发条件**：可主动召唤（每次≤5分钟）或被动失控
   - **表现**：左眼纯黑，左手青白化，布满黑色经络，理智与本能撕裂
   - **当前掌控度**：30%（第18章状态）
   - **副作用**：每次使用侵蚀度+5%，体温下降，记忆碎片化
4. **鬼眼视觉**：能看穿灵怨流动，识别煞灵核心（第18章新觉醒）

**关键道具**
- **《镇灵谱》真本**：爷爷用血写就，夹层有遗书，末页有指甲刻字
- **问玉玦碎片**：镇压血咒的古玉已碎，残留灵契碎片3/4
- **星纹骨图**：萨满教星图，指向滇池影都，需血咒激活
- **鬼手**：非自愿获得，既是诅咒也是武器

**人物弧光**
- **起始**：恪守祖训的理论派，洁癖学者，实战经验为零
- **转变**：问玉玦碎裂后被迫接纳血咒，从"镇压者"变为"掌控者"
- **当前（第18章）**：可主动召唤鬼手，领悟"控之"法门，开始接受九幽之力而非单纯镇压
- **未来**：将在滇池影都面临"控之"还是"从之"的终极选择

**关系网**
- **陆航**：发小，从雇佣关系到过命交情。唯一敢吐槽他的人
- **云舒**：神秘盟友，互相利用。她知晓爷爷往事，但有所保留
- **爷爷姜穆**：失踪的引路人，其留下的线索是最大谜团
- **苍龙会**：未知敌人，似乎与爷爷失踪有关

**当前状态**（第18章）
- **血咒侵蚀度**：30%（暂时稳定，可主动召唤鬼手）
- **身体状态**：左肩刀伤未愈，鬼手偶发刺痛，体温持续偏低
- **心理负担**：恐惧血咒失控，对云舒的真实目的存疑，但开始接受"控之"理念
- **装备更新**：集齐3/4灵契碎片，获得星纹骨图（显示37个岔路）
- **目标**：沿地下河寻找苍龙会基地，解开爷爷失踪之谜

### 陆航 (Lu Hang) - 前特种兵/爆破专家

**基础信息**
- **年龄**：32岁
- **身高**：185cm
- **体重**：95kg
- **外号**：陆胖子（姜辰专用）
- **外貌**：壮硕，寸头，满脸横肉但眼神清澈，常穿战术背心
- **身份**：退役特种兵 / 长途货车司机（负债版）

**性格特征**
- **核心特质**：豪爽、乐观、贪财但有底线，团队气氛组
- **优点**：实战经验丰富，爆破与射击技术一流，绝对忠诚，危机时刻敢拼命
- **缺点**：冲动，话痨，对超自然现象接受度低，习惯性吐槽
- **口头禅**："我操" / "加钱" / "这不科学" / "学者你行不行啊"
- **恐惧**：欠债不还，兄弟死在自己面前

**背景故事**
- 前某军区特种作战大队爆破手，退役后因帮战友担保背上300万债务
- 被高利贷追得走投无路时，姜辰开出"三十万，预付一半"的价码
- 起初以为只是普通考古，发现真相后已无法脱身
- 发小姜辰是他唯一信任的人，愧疚于当年没能保护他

**核心动机**
- **表层**：赚钱还债，恢复自由身
- **深层**：保护姜辰，弥补当年未能守护兄弟的遗憾
- **潜在**：在超自然世界中寻找自己的价值定位

**特殊能力**
1. **现代战术**：C4爆破、枪械改造、战场急救
2. **工程直觉**：能用普通材料制作土制武器（燃烧瓶、陷阱）
3. **负重能力**：能背负超过50kg装备在恶劣环境长途跋涉
4. **战斗本能**：对危险有敏锐直觉，但面对灵怨时经常失灵

**关键道具**
- **M249班用机枪**：改装版，使用钨芯穿甲弹（对沙鳞蚰无效）
- **C4炸药**：共携带8块，玄甲墓中已用6块，剩余2块
- **工兵铲**：钨钢刃口，可近战可挖掘
- **卫星电话**：意外在墓中有信号，收到催债电话
- **战术背心**：装满各种工具，被沙鳞蚰咬破后损失40%物资

**人物弧光**
- **起始**：纯粹为钱，把姜辰当金主，对超自然现象完全不信
- **转变**：第12章为姜辰挡枪，意识到兄弟比钱重要
- **当前（第18章）**：接受超自然世界存在，但坚持用"物理"解决问题，主动承担危险任务
- **未来**：将在滇池影都面临"科学"与"灵异"的认知崩塌

**关系网**
- **姜辰**：发小，从金主到兄弟。唯一会听他吐槽的人
- **云舒**：从怀疑到信任。欣赏她的身手，但不满她的神秘
- **催债人**：现实中的压力源，让他不得不继续冒险
- **战友**：债务源头，也是他坚守"义气"的原因

**当前状态**（第18章）
- **身体状态**：左肩脱臼已复位，背包损失40%物资，在矿洞入口与姜辰云舒汇合
- **心理状态**：接受超自然世界存在，但坚持用"物理"解决问题，对兄弟的责任感超越金钱
- **装备更新**：C4炸药已耗尽（第29章用完），雷明顿M870剩2发鹿弹 + 2枚手雷
- **债务**：剩余180万，但动机已从"为钱"转为"为兄弟"
- **目标**：保护姜辰前往苍龙会基地，寻找出路

### 云舒 (Yun Shu) - 神秘考古学家/复仇者

**基础信息**
- **年龄**：29岁
- **身高**：172cm
- **体重**：55kg
- **外貌**：冷艳，黑衣，常年戴战术面罩，只露出一双锐利的眼睛。右臂骨折后改用背带固定
- **身份**：考古学博士 / 苍龙会追杀对象 / 姜家秘密守护者

**性格特征**
- **核心特质**：高冷、敏锐、神秘，话少但每句都直击要害
- **优点**：身手顶尖（尤其短刀与暗器），知识渊博（对镇灵师体系了如指掌），判断力极强
- **缺点**：不信任他人，习惯单打独斗，复仇心重，为达目的不择手段
- **口头禅**："……"（沉默） / "蠢货" / "这不重要" / "听我的"
- **恐惧**：姜辰血咒失控，苍龙会抢先找到九幽灵，父亲死得毫无价值

**背景故事**（渐进式揭露）
- **阶段一（第13-20章）**：表面身份是考古学家，装备精良，对玄甲墓异常熟悉
- **阶段二（第21-34章）**：父亲云深是考古学家，二十年前在玄甲墓失踪
- **阶段三（第35-40章）**：舍命救姜辰，动机是为利用他的血脉寻找父亲遗物
- **阶段四（第41-48章）**：矿道日记暗示父亲当年是姜辰爷爷的助手
- **阶段五（第49-50章）**：真相揭露——父亲被苍龙会杀害，她需要姜辰的血脉复仇

**核心动机**
- **表层**：寻找父亲失踪的真相，取回遗物
- **深层**：向苍龙会复仇，摧毁他们唤醒九幽灵的计划
- **潜在**：保护姜辰，因为他是唯一能与九幽灵对抗的镇灵师

**特殊能力**
1. **古武传承**：短刀术（已断）、暗器、单手格斗术
2. **考古知识**：精通萨满教、古滇国、痋术等冷门历史
3. **灵怨嗅觉**：能闻到灵怨的味道（铁锈混腐肉），问玉玦碎后成为团队唯一感知手段
4. **战术规划**：擅长设计陷阱与逃生路线，但习惯独自行动

**关键道具**
- **黑短刀×2**：爷爷姜穆所制，有灵怨回路，一把已断，一把钉在姜辰肩上唤醒他
- **考古仪器**：比陆航的GPS更精准，能检测地煞流动
- **灵契碎片**：父亲留下的半块碎片，与姜辰的碎片共鸣
- **静音装备**：静音靴、消音布，为潜入墓中专门准备

**人物弧光**
- **起始**：神秘第三方，对姜辰团队若即若离，完全隐藏目的
- **转变**：第36章舍命抢夺头盔，暴露真实目的（利用姜辰血脉）
- **当前（第18章）**：承认需要姜辰，父亲被苍龙会杀害的秘密部分揭露，从"利用"转向"守护"但嘴上不承认
- **未来**：将在滇池影都面临"复仇"与"守护"的终极抉择

**关系网**
- **姜辰**：利用对象→合作者→命运共同体。她需要他的血脉，但也在保护他
- **陆航**：从鄙视到认可。欣赏他的忠诚，但保持距离
- **父亲云深**：精神支柱，复仇的源头
- **爷爷姜穆**：父亲口中的"黑衣镇灵师"，她一直在寻找他的踪迹
- **苍龙会**：杀父仇人，她的一切行动都围绕对抗他们

**当前状态**（第18章）
- **身体状态**：右臂骨折未愈，左臂新增三道割伤（第38章战斗），失血约300ml
- **心理状态**：复仇火焰与保护姜辰的责任感交织，对姜辰态度软化但保持距离
- **秘密揭露**：父亲是姜辰爷爷助手，被苍龙会杀害（第47章日记揭示）
- **装备更新**：仅剩一把断刀，考古仪器在战斗中损坏
- **目标**：引导姜辰前往苍龙会基地，寻找父亲遗物，同时防止血咒失控

## 重要配角

### 姜守仁（爷爷）- 白衣镇灵师 / "背叛者"
- **年龄**：失踪时58岁（20年前）
- **身份**：镇灵师姜家第46代传人，姜辰的爷爷，"白衣"一脉最后传人
- **外貌**：清瘦，白发，常年穿白色长衫，气质儒雅如学者
- **性格**：表面恪守祖训"只镇不取"，实则内心渴望掌控九幽灵之力
- **核心秘密**：20年前发现九幽灵真相后，从"镇压者"变为"夺取者"
- **关键行为**：
  - 将《镇灵谱》真本和问玉玦留给姜辰，诱导他下墓
  - 将灵契碎片分散（3块在玄甲墓，1块在水煞泉眼）镇压苍龙会
  - 在玄甲墓与弟弟姜守玄反目，引爆"九宫锁龙阵"将其困死
  - 战胜姜守玄后未回地面，下落成谜
- **与姜辰关系**：既是精神导师，也是欺骗者。他算到了姜辰的血咒，故意布下这个局
- **当前状态**：生死不明，但云舒父亲日记暗示"白衣先生没上来，他封了出口"
- **战力**：巅峰时期可单手镇压将军煞，血咒掌控度超过80%

### 姜守玄（二爷爷）- 黑衣镇灵师 / 苍龙会创始人
- **年龄**：失踪时54岁（20年前）
- **身份**：姜守仁的弟弟，"黑衣"一脉开创者，苍龙会创始人
- **外貌**：与姜守仁相似但更阴鸷，常年穿黑色长衫，眼神狂热
- **性格**：激进、偏执，认为"镇压是懦弱，掌控才是未来"
- **核心信念**：九幽灵不是诅咒，是钥匙，是打开更高层次力量的钥匙
- **关键行为**：
  - 20年前创建苍龙会，聚集一批"进化论"追随者
  - 在玄甲墓设下"苍龙之门"陷阱，专门吞噬同源血咒
  - 被姜守仁击败后，怨念被水煞吞噬，成为"摆渡人"（水煞·姜守玄）
  - 死后仍在"守卫"萨满祭坛，引诱姜家后人下水
- **与姜辰关系**：血缘上的二爷爷，实际上的敌人。他的失败是姜辰的前车之鉴
- **当前状态**：已死亡，怨念被水煞吞噬，成为黑水湖的"摆渡人"守卫
- **战力**：生前与姜守仁不相上下，死后成为水煞集合体，拥有"鬼手"和"高压水箭"能力

### 云深（云舒父亲）- 考古学家/姜守仁助手
- **年龄**：死亡时42岁（20年前）
- **身份**：考古学博士，姜守仁的助手兼好友，云舒的父亲
- **外貌**：斯文，戴圆框眼镜，常年穿考古工装
- **性格**：理性、谨慎，对镇灵师体系充满好奇但不盲目崇拜
- **关键行为**：
  - 20年前跟随姜守仁下玄甲墓，负责考古记录
  - 在地下河转角石壁留下血字警告"姜守玄之墓，别信姜守仁"
  - 被苍龙会追杀，临死前将半块灵契碎片交给女儿云舒
  - 日记中记载了九幽灵的"回响"现象和姜家两脉决裂真相
- **与姜辰关系**：间接引路人，他的血字和遗物指引姜辰前进
- **当前状态**：确认死亡，遗体在萨满祭坛附近被发现
- **遗物**：半块灵契碎片、考古笔记（在滇池影都）、血字刻痕

### 苍龙会（组织）- 敌对势力
- **性质**：与镇灵师对立的神秘组织，认为灵怨是"进化"关键
- **创始人**：姜守玄（20年前）
- **现任首领**："鬼先生"（第二卷登场）
- **核心成员**：黑衣镇灵师后裔，主张"以煞制煞"
- **手段**：破坏封印，唤醒九幽灵，追求永生
- **目标**：集齐所有灵契碎片，打开九幽灵本体封印
- **与姜家关系**：
  - 20年前追杀姜守仁，导致云深死亡
  - 一直在寻找姜家血脉，认为只有姜家血能激活九幽灵
  - 第二卷将在滇池影都设下陷阱，等待姜辰

### 苍龙会（组织）- 敌对势力
- **性质**：与镇灵师对立的神秘组织，认为灵怨是"进化"关键
- **手段**：破坏封印，唤醒九幽灵，追求永生
- **已知成员**："鬼先生"（第二卷登场）
- **目标**：集齐所有灵契碎片，打开九幽灵本体封印

## 创作要点提醒

### 姜辰塑造要点
- **洁癖细节**：墓道中用酒精湿巾擦手，避开骸骨，对陆航的邋遢忍无可忍
- **学者气质**：随时引用《镇灵谱》，用科学原理解释灵异现象（虽然经常失败）
- **血咒恐惧**：每次鬼手发作后，会偷偷检查身体异变，害怕被陆航发现
- **成长标志**：从"我不能碰"到"我必须碰"，从"理论"到"控之"

### 陆航塑造要点
- **贪财细节**：每得到一件古董都估价，被云舒鄙视；但为姜辰花钱从不犹豫
- **物理执念**：坚持用C4、子弹解决一切，直到发现对将军煞无效
- **义气表现**：第十二章为姜辰挡枪是转折点，之后从"为钱"变"为兄弟"
- **成长标志**：从"这不科学"到"去他妈的科学，学者说啥就是啥"

### 云舒塑造要点
- **神秘感**：话只说三分，永远留一手，连救姜辰都说是"利用"
- **专业细节**：装备都是定制，对墓中机关了如指掌，但从不解释来源
- **复仇火焰**：提到苍龙会时眼神会变化，但被姜辰问起又立刻冷下来
- **成长标志**：从"我一个人就够了"到"你们跟上"，从"利用"到"守护"
